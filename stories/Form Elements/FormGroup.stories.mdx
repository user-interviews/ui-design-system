# Input Fields Anatomy

Input fields let users enter and edit text. These are the variants available in the design system.

## Placeholder Text

- Used to give the user an idea of what sort text they should include here. Does not carry over as default text if no other text is added, so if left blank will produce an error. 

<Preview>
<Story name="Input Fields">
<FormGroup id="default">
  <InputComponent id="default-input" name="default" placeholder="Placeholder text" />
</FormGroup>
</Story>
</Preview>

## Required

- To indicate that a field is required, display "(Required)" next to the label
- If some fields are required, indicate all required ones
- If most fields are required, indicate optional fields by displaying "(Optional)" next to the label
