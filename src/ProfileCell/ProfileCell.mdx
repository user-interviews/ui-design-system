# Profile Cell
Displays information about a user which can include an avatar, name, email, phone number, timezone (or other relevant pieces of information).

## Anatomy

- **Avatar:** Displays photo or initials of a user
- **Name:** First and last name 
- **Email** 
- **Phone number** 
- **Timezone** 

## Best practices
- Content inside cards should be left-aligned. Deviations are left up to designer's discretion.
