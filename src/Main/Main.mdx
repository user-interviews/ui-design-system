## When should you use Main?

* Every document page needs at least one HTML `<main>` landmark to ensure there is a navigation point to the primary content of the page

## Accessibility guidance

* Content inside `<Main>` should be unique to the document. It should not contain any content that is repeated across documents such as sidebars, navigation links, copyright information, site logos, etc.
* There must not be more than one `<Main>` element in a document.
* The `<Main>` element must NOT be a descendant of an `<article>`, `<aside>`, `<footer>`, `<header>`, or `<nav>` element.
* Use Main in order to avoid the following moderate accessibility errors: 
  * [axe-core-4.3: All page content must be contained by landmarks](https://dequeuniversity.com/rules/axe/4.3/region)
  * [axe-core-4.4: Page must have one main landmark](https://dequeuniversity.com/rules/axe/4.4/landmark-one-main?application=AxeChrome)
