# Accordion

The accordion is a graphical control element comprising a vertically stacked list of items, such as labels or thumbnails. Each item can be "expanded" or "collapsed" to reveal the content associated with that item.

## Anatomy

- **Icon** (optional): Decorative icon used as visual reinforcement.
- **Title:** Describes what is contained within the accordion.
- **Helper text:** Provides context on how to use the accordion (e.g. displays whether it contains required inputs).
- **Header:** Clicking anywhere within this top container should cause the accordion to expand or collapse.
- **Label (optional):** Displays secondary information about the accordion (e.g. the # of selections made).
- **Chevron:** Indicates whether the accordion is expanded or collapsed.
- **Body:** Where all of the accordionâ€™s content is contained. Only visible when the accordion is expanded.
